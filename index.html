Block Templates

Block Templates

What are Block Templates?

Block Templates are a set of predefined blocks for content defined in the editor and stored in post_content.

Block Templates

Why Use Block Templates?

Block templates allow theme builders to provide the admin user with guiderails to add content.


They make it easier for the user to get content on the page by limiting the design choices they need to make.

Beyond preserving the integrity of the design, the primary role of block templates provides better UX/UI.
Block Templates
This would be pretty tedious for user to add all of these blocks each time, so we'll build a block template allowing them to quickly get started.
Block Templates

Block Templates
vs Page Templates

Block Templates impose structure on post content,
they are unrelated to page templates found in themes.

They are not what you would see in the dropdown of templates under Documents in the Inspector Controls. You might want to add block templates to a page template or post template, and we will see a few ways this can be accomplished as we look at each method used to create block templates. So let's get started.
Block Templates

Block Templates
vs Block Patterns

Block Patterns are a layout grid that allow an admin user to quickly get a layout.

These are just coming into use, with new patterns being developed.

WordPress Block Editor Handbook

Block Patterns

How they work

The register_block_pattern function receives the name of the pattern as the first argument and an array describing properties of the pattern as the second argument.

The properties of the style array must include name and label:

title: A human-readable title for the pattern.

content: Raw HTML content for the pattern.

So Meta

The rise of the meta box came about from the limitation of structuring content in the Classic WordPress editor.

This was a powerful tool allowing theme creators to customize how admin users added content to their site that could be both stored for use in other places as well as displayed on the front end. Using tools like ACF, theme authors could make it easier to put together the Real Estate Lising.

What is a Block-Based Theme?

A WordPress theme created with HTML files built entirely with blocks. Block-based themes are minimal html structures that provide the containers for rendering blocks in the browser.

Full Site Editing

What is Full Site Editing?

An evolution of the editing experience allowing all areas of the site including header, footer, navigation to be edited from the Admin Dashboard. You can test it from the Site Editor Beta under the experimental flag in the Gutenberg Plugin.

It creates a mini sandbox of a proposed version of the editor. It's not a part of Core and should not be used on production.
